# 开始

非常欢迎您的加入！

此文档使用 [vitepress](https://vitepress.vuejs.org/) 构建，详细使用方法请参考官方网站。下面简单介绍一下 `docs` 模块的目录结构

## 目录结构

```text
/docs
├── _data # 数据目录。为什么前面要加下划线？vite官方的vitepress就是这个名字
│   └── team.js # 团队成员
├── development # 开发
│   ├── add-event-action.md
│   ├── ...
├── guide # 指南
│   ├── component.md
│   ├── ...
├── public # 资源目录
│   ├── icon.svg
│   ├── ...
├── index.md # 首页
├── package.json
├── team.md # 团队展示页面
├── tsconfig.json
└── vite.config.ts
```
